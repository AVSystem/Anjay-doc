9.1. Migrating from Anjay 1.16Â¶

Anjay 2.x introduced a lot of new features compared to the 1.x branch. A lot of the changes are related to the introduction of LwM2M TS 1.1 support in the commercial version. To fully support all the new features, a significant number of breaking changes have been introduced.

This chapter is a summary of changes, to guide users of Anjay 1.16 with migrating to the current version.
