◆ anjay_persistence_bytes()

int anjay_persistence_bytes ( anjay_persistence_context_t *  ctx,
uint8_t *  buffer,
size_t  buffer_size 
)

Performs operation (depending on the ctx) on byte sequence.

On restore context behavior:

  • buffer is a pointer to the user-allocated buffer.
  • buffer_size is a size of the user-allocated buffer.
  • If the data cannot be fit into the buffer, then an error is returned.

On persist context behavior:

  • buffer is a pointer to the user-allocated buffer.
  • buffer_size is the amount of bytes to store.
  • If the data cannot be stored, then an error is returned.

On ignore context behavior:

  • buffer is optional, might be NULL.
  • buffer_size is the amount of bytes to be ignored.

Example usage:

char buffer[1024];
// Some logic that fills the buffer
...
uint32_t buffer_size = ...;
// Store buffer size so that it can be restored later.
int retval = anjay_persistence_u32(persist_ctx, &buffer_size);
if (retval) {
return retval;
}
// Store the buffer itself.
return anjay_persistence_bytes(persist_ctx, buffer, buffer_size);
Parameters
ctxContext that determines actual operation.
bufferPointer to the user-allocated buffer or NULL.
buffer_sizeSize of the user-allocated buffer or 0.
Returns
0 in case of success, negative value in case of failure.
